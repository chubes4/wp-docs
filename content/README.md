# Content

This directory holds generated and reviewed documentation content.

Planned lanes:

- `raw/` — disposable generated drafts.
- `user/` — user-facing documentation.
- `developer/` — developer-facing guides.
- `reference/` — source-derived reference documentation.
- `internals/` — Core internals and contributor-oriented architecture notes.

Generated drafts should be reproducible. Reviewed content should keep provenance metadata in `provenance/`.
