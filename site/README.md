# Site

This directory is reserved for the WordPress publishing surface.

The target direction is a WordPress.org-compatible site implementation, likely including:

- WordPress import/export tooling.
- A docs-oriented WordPress block theme.
- Tailwind-powered build tooling if it remains compatible with the target deployment path.
- Navigation and search experiments.

The site implementation should remain separate from generated content so docs can be regenerated without coupling generation to presentation internals.
